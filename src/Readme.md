第一部分：项目概述

1.1 项目背景

我们要在中东（阿联酋、沙特）卖中国紫砂壶，需要做一个电商网站：

• 卖现货紫砂壶
• 核心：支持客户在线定制紫砂壶（选壶型、刻字、图案）
• 会员推荐系统
• 支持多语言（中文/英文/阿拉伯语）
• 支持多币种（AED/SAR/USD）

1.2 技术栈
前端：Next.js 14+ (App Router)
样式：Tailwind CSS + shadcn/ui
后端：Next.js API Routes
数据库：PostgreSQL (Neon/Supabase)
图片存储：Cloudflare R2
支付：Stripe + PayPal
部署：Vercel
───

模块一：商品系统

1.1 商品列表页 /products

功能描述：
用户浏览商品的主页面，支持分类筛选、搜索、排序、分页。

详细需求：
UI布局：
├── 顶部：分类筛选栏（横向滚动）
│   ├── 全部
│   ├── 紫砂壶
│   ├── 茶杯
│   ├── 茶叶罐
│   └── 套装
├── 左侧：筛选面板
│   ├── 价格区间（滑块：0-5000 AED）
│   ├── 泥料（多选：紫泥/红泥/段泥）
│   ├── 容量（多选：100ml/150ml/200ml/300ml/500ml）
│   ├── 壶型（多选：石瓢/西施/井栏/仿古/供春/水平/掇球）
│   └── 库存状态（有货/预售）
├── 主体：商品网格（3列）
│   ├── 商品卡片（图片/标题/价格/销量/评分）
│   └── 无限滚动加载
└── 底部：分页控件

交互逻辑：
1. 点击分类 → 刷新商品列表
2. 选择筛选条件 → 即时过滤
3. 点击商品卡片 → 跳转商品详情
4. 下拉加载更多 → 分页加载
API接口：
GET /api/products
  参数：
    - page: number (默认1)
    - limit: number (默认12)
    - category: string (分类)
    - search: string (搜索词)
    - minPrice/maxPrice: number
    - materials: string[] (泥料)
    - capacities: number[] (容量)
    - sort: 'newest' | 'price-asc' | 'price-desc' | 'sales'
  返回：
    - products: Product[]
    - total: number
    - page: number
    - totalPages: number
